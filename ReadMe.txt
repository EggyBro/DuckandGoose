# DuckandGoose
Main Project Repository for DigiPen GAM200

Overview ---------------------------------------------------------------------------------------------
Exomata is a fast-paced hack-and-slash, side-scroller game revolving around a run-down 
mech who awakens with no recollection of the past and embarks on a journey through the 
post-apocalyptic world to recover his lost memories and rediscover who he really is.
------------------------------------------------------------------------------------------------------

Usage ------------------------------------------------------------------------------------------------
- Before starting the game, build the exomata engine first.
- Engine requires the latest OpenGL version 4.5 to operate.
- Only compatible with dedicated graphics card.
------------------------------------------------------------------------------------------------------

OpenGL 4.5 configuration (if using intel UHD graphics) -----------------------------------------------
- Right Click and go into nVidia Control Panel
- Click on tab "manage 3D settings"
- Add "Exomata.exe" to list of programs
- Change Preferred Graphics Processor to Dedicated graphics card
- Click Apply
------------------------------------------------------------------------------------------------------

Controls ---------------------------------------------------------------------------------------------
- Movement using W,A,S,D
- Attack using Left Mouse Click
- Press ESC for pause state when LevelEditor is off.
- Press P to toggle on and off LevelEditor.
------------------------------------------------------------------------------------------------------
